# Jaypore CI

> A CI system that sounds ancient and powerful.
> Like the city of Jaypore.

## Usage

```bash
# Install once
curl https://raw.githubusercontent.com/theSage21/jaypore_ci/main/setup.sh | bash
git add -Av
git commit -m 'Add Jaypore CI'
git push origin


# Trigger CI whenver you push
git push origin
```
